# 🏦 Bank Management System

A command-line based Bank Management System developed using **Python** and **MySQL**. The project enables secure user registration, authentication, account management, balance enquiry, deposits, withdrawals, and transaction tracking.

## 🚀 Features
- User Registration & Login
- Unique Account Number Generation
- Balance Enquiry
- Cash Deposit & Withdrawal
- Transaction Management
- MySQL Database Integration

## 🛠️ Tech Stack
- Python
- MySQL
- MySQL Connector

python main.py
